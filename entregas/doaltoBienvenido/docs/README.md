# Algoritmos de Ordenación - Guía de Traza Detallada

## 📂 Estructura de Archivos

Puedes acceder a los códigos fuente directamente a través de los siguientes enlaces:

### 1. Inserción (Insertion Sort)
* **[Inserción Iterativa](../src/InsertionSort.java)**: Implementación clásica utilizando bucles `for` y `while`.
* **[Inserción Recursiva](../src/InsertionSortRecursivo.java)**: Versión que utiliza recursión para reducir el tamaño del problema antes de insertar.

### 2. Mezcla (Merge Sort)
* **[Merge Sort Iterativo](../src/MergeSortIterativo.java)**: Ordenación por mezcla utilizando un enfoque *bottom-up* con bloques de tamaño creciente.
* **[Merge Sort Recursivo](../src/MergeSortRecursivo.java)**: Implementación *top-down* que divide el array sistemáticamente hasta alcanzar el caso base.

### 3. Ordenación Rápida (Quick Sort)
* **[Quick Sort](../src/QuickSort.java)**: Algoritmo basado en la elección de un pivote y particionamiento del array, con traza de las ramas izquierda y derecha.
